
# Stock Portfolio Tracker

## Description
The Stock Portfolio Tracker is a Python-based application used to manage and track stock investments. This project allows users to add stocks, view portfolio details, calculate total investment value, view available stocks, and save portfolio information efficiently.

## Objectives
- To develop a simple stock portfolio tracking system using Python
- To understand the concept of stock investment management
- To practice Python programming concepts such as loops, dictionaries, conditions, and file handling
- To calculate and track total investment value
- To create a user-friendly menu-driven application

## Features
- Add Stock
- View Portfolio
- View Total Investment
- View Available Stocks
- Save Portfolio
- Exit Program

## Technologies Used
- Python
- VS Code
- GitHub

## How to Run
1. Install Python
2. Open the project folder in VS Code
3. Run the Python file:
```bash
python stock_portfolio_tracker.py
